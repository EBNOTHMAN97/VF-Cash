# 📚 دليل الاستخدام السريع - نظام فودافون كاش

## 🎯 البداية السريعة

### الخطوة 1️⃣: افتح التطبيق
عند فتح التطبيق لأول مرة، ستجد صفحة فارغة مع خيارين:
- **+ إضافة محفظة جديدة** - لإضافة محفظة يدوياً
- **🎲 بيانات تجريبية** - لإضافة 6 محافظ تجريبية تلقائياً (مناسب للتجربة)

### الخطوة 2️⃣: إضافة محافظك
اضغط على "إضافة محفظة جديدة" وأدخل البيانات:

```
مثال:
- الاسم: محفظة العمل
- رقم الهاتف: 01012345678
- الحد اليومي: 5000 جنيه
- الحد الشهري: 50000 جنيه
- الرصيد الحالي: 2500 جنيه
- اللون: اختر أي لون مميز
```

### الخطوة 3️⃣: إضافة معاملات
عند إجراء أي عملية إرسال أو استلام:

1. اضغط **+ إضافة معاملة**
2. اختر المحفظة
3. اختر نوع العملية:
   - **إرسال (خصم)** ← عند دفع أموال
   - **استلام (إضافة)** ← عند استلام أموال
4. أدخل المبلغ والوصف
5. اضغط "إضافة"

## 📊 فهم البطاقات

كل محفظة تعرض:

### 💵 الرصيد الحالي
```
الرصيد الحالي: 2,500.00 ج.م
```

الحد اليومي: 3,200.00 / 5,000.00 ج.م

## ⚡ أمثلة عملية

### مثال 1: تسجيل تحويل للعميل
```
المحفظة: محفظة العمل
النوع: إرسال (خصم)
المبلغ: 500
الوصف: تحويل للعميل أحمد - فاتورة #123

النتيجة:
✅ الرصيد: 2500 ← 2000 ج.م
✅ الحد اليومي: +500 ج.م
✅ الحد الشهري: +500 ج.م
```

### مثال 2: استلام دفعة من عميل
```
المحفظة: محفظة العمل
النوع: استلام (إضافة)
المبلغ: 1000
الوصف: دفعة من العميل محمد

النتيجة:
✅ الرصيد: 2000 ← 3000 ج.م
❌ الحدود: لا تتأثر (الاستلام لا يُحسب)
```

### مثال 3: متابعة محفظة معينة
```
1. اضغط على بطاقة المحفظة
2. ستظهر جميع معاملات هذه المحفظة فقط
3. اضغط "عرض الكل" للعودة لجميع المعاملات
```

## 🎨 الألوان والمؤشرات

### ألوان شريط التقدم:
- 🟢 **0-70%**: كل شيء تمام ✓
- 🟡 **70-90%**: احذر! قريب من الحد ⚠️
- 🔴 **90-100%**: خطر! وصلت للحد 🚫

### أيقونات المعاملات:
- 📤 **إرسال**: لون أحمر (خصم من الرصيد)
- 📥 **استلام**: لون أخضر (إضافة للرصيد)

## 💡 نصائح مهمة

### ✅ افعل:
- سجل كل معاملة فور حدوثها
- راقب شريط التقدم باستمرار
- استخدم الوصف لتوضيح كل معاملة
- اختر لون مميز لكل محفظة

### ❌ لا تفعل:
- لا تنسى تسجيل المعاملات
- لا تعتمد على الذاكرة - سجل فوراً
- لا تتجاهل التحذيرات الصفراء/الحمراء

## 📱 الوصول السريع

### من الموبايل:
1. افتح المتصفح
2. ادخل على الرابط
3. اضغط "Add to Home Screen"
4. الآن التطبيق على شاشتك الرئيسية!

## 🔄 تحديث البيانات

### متى تُحدّث الحدود؟
- **الحد اليومي**: يُعاد كل يوم الساعة 00:00
- **الحد الشهري**: يُعاد أول كل شهر ميلادي

### كيف يتم الحساب؟
```
الحد اليومي = مجموع كل عمليات "الإرسال" اليوم
الحد الشهري = مجموع كل عمليات "الإرسال" هذا الشهر
الرصيد = الرصيد السابق + الاستلام - الإرسال
```

## 🆘 حل المشاكل الشائعة

### المشكلة: البيانات لا تظهر
**الحل**: اضغط F5 لتحديث الصفحة

### المشكلة: الحدود لا تتحدث
**الحل**: تأكد أن نوع المعاملة "إرسال" وليس "استلام"

### المشكلة: الرصيد خطأ
**الحل**: راجع جميع المعاملات وتأكد من صحة المبالغ

## 🎓 سيناريو عملي كامل

تخيل أنك تدير محل وتستخدم محفظتين:

### اليوم الأول:
```
⚙️ إعداد المحافظ:
✓ محفظة المحل (حد يومي: 10,000 ج.م)
✓ محفظة شخصية (حد يومي: 5,000 ج.م)

💰 الصباح - استلام من العميل أحمد:
📥 استلام | محفظة المحل | 2,000 ج.م
الرصيد: 0 → 2,000 ج.م

🛒 الظهر - دفع للمورد:
📤 إرسال | محفظة المحل | 1,500 ج.م
الرصيد: 2,000 → 500 ج.م
الحد اليومي: 1,500 / 10,000 (15%)

💳 المساء - تحويل شخصي:
📤 إرسال | محفظة شخصية | 800 ج.م
الحد اليومي: 800 / 5,000 (16%)
```

### مراجعة نهاية اليوم:
```
محفظة المحل:
✓ الرصيد: 500 ج.م
✓ استهلاك الحد اليومي: 15% (آمن 🟢)

محفظة شخصية:
✓ الرصيد: حسب المعاملات
✓ استهلاك الحد اليومي: 16% (آمن 🟢)
```
