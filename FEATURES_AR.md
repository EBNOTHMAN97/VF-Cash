## 💳 تتبع المعاملات

### أنواع المعاملات

#### 1. الإرسال (Send) 📤
**متى تستخدمه:**
- دفع لمورد
- تحويل لعميل
- دفع فاتورة
- سحب نقدي

**التأثير:**
```
الرصيد: ⬇️ ينقص
الحد اليومي: ⬆️ يزيد المستخدم
الحد الشهري: ⬆️ يزيد المستخدم
```

**مثال:**
```
قبل: رصيد 5000 ج.م، حد يومي 2000/10000
معاملة: إرسال 500 ج.م
بعد: رصيد 4500 ج.م، حد يومي 2500/10000
```

#### 2. الاستلام (Receive) 📥
**متى تستخدمه:**
- استلام من عميل
- إيداع نقدي
- تحويل من محفظة أخرى

**التأثير:**
```
الرصيد: ⬆️ يزيد
الحد اليومي: ➖ لا يتأثر
الحد الشهري: ➖ لا يتأثر
```

**مثال:**
```
قبل: رصيد 1000 ج.م، حد يومي 3000/10000
معاملة: استلام 2000 ج.م
بعد: رصيد 3000 ج.م، حد يومي 3000/10000 (لم يتغير)
```

## 📊 الحدود والقيود

### كيف يتم الحساب؟

#### الحد اليومي:
```javascript
// يبدأ من الساعة 00:00 كل يوم
المستخدم = مجموع كل معاملات "الإرسال" من 00:00 حتى الآن
المتبقي = الحد اليومي - المستخدم
النسبة = (المستخدم / الحد اليومي) × 100
```

**مثال يوم كامل:**
```
الساعة 09:00 - إرسال 500 ج.م
المستخدم: 500 / 5000 = 10%

الساعة 12:00 - إرسال 1000 ج.م
المستخدم: 1500 / 5000 = 30%

الساعة 15:00 - استلام 2000 ج.م
المستخدم: 1500 / 5000 = 30% (لم يتغير)

الساعة 18:00 - إرسال 2500 ج.م
المستخدم: 4000 / 5000 = 80% ⚠️
```

#### الحد الشهري:
```javascript
// يبدأ من اليوم الأول من كل شهر
المستخدم = مجموع كل معاملات "الإرسال" هذا الشهر
المتبقي = الحد الشهري - المستخدم
النسبة = (المستخدم / الحد الشهري) × 100
```

### نظام الألوان التحذيري

#### 🟢 المنطقة الآمنة (0-70%)
```
الحالة: طبيعي
الإجراء: لا شيء - استمر في العمل
مثال: 3,500 / 5,000 = 70%
```

#### 🟡 منطقة التحذير (70-90%)
```
الحالة: احذر!
الإجراء: راقب معاملاتك بعناية
مثال: 4,250 / 5,000 = 85%
نصيحة: قد تحتاج لاستخدام محفظة أخرى
```

#### 🔴 المنطقة الحرجة (90-100%)
```
الحالة: خطر!
الإجراء: استخدم محفظة بديلة فوراً
مثال: 4,750 / 5,000 = 95%
تحذير: قد لا تتمكن من إجراء معاملات جديدة
```

## 🎨 واجهة المستخدم

### التصميم العربي

#### خصائص الواجهة:
```
✓ اتجاه RTL (من اليمين لليسار)
✓ خطوط عربية واضحة
✓ أيقونات Emoji للتوضيح
✓ ألوان فودافون (أحمر وأبيض)
✓ تصميم متجاوب (موبايل + كمبيوتر)
```

### التفاعلية

#### الأزرار:
| الزر | الوظيفة | اللون |
|-----|---------|-------|
| + إضافة محفظة | إضافة محفظة جديدة | 🔴 أحمر |
| 🎲 بيانات تجريبية | إضافة محافظ تجريبية | 🔵 أزرق |
| + إضافة معاملة | تسجيل معاملة جديدة | 🟢 أخضر |
| عرض الكل | عرض جميع المعاملات | ⚫ رمادي |

#### النوافذ المنبثقة (Modals):
```
1. نافذة إضافة محفظة:
   - نموذج بـ 6 حقول
   - زر "إضافة" - زر "إلغاء"

2. نافذة إضافة معاملة:
   - نموذج بـ 4 حقول
   - قائمة منسدلة للمحافظ
   - اختيار نوع المعاملة
```

### الاستجابة (Responsive)

#### على الموبايل:
```
📱 شاشة صغيرة (< 768px):
✓ عمود واحد للمحافظ
✓ جدول قابل للتمرير أفقياً
✓ أزرار كاملة العرض
```

#### على التابلت:
```
📱 شاشة متوسطة (768px - 1024px):
✓ عمودين للمحافظ
✓ جدول كامل العرض
```

#### على الكمبيوتر:
```
💻 شاشة كبيرة (> 1024px):
✓ 3 أعمدة للمحافظ
✓ جدول كامل مع جميع الأعمدة
```
