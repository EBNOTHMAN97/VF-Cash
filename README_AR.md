# 📱 نظام متابعة محافظ فودافون كاش

تطبيق ويب متكامل لمتابعة وإدارة محافظ فودافون كاش مع تتبع الحدود اليومية والشهرية.

## ✨ المميزات

- 💰 **إدارة 6 محافظ** (أو أكثر) من فودافون كاش
- 📊 **متابعة الحد اليومي** - تتبع المبالغ المصروفة يومياً مقابل الحد المسموح
- 📈 **متابعة الحد الشهري** - تتبع المبالغ المصروفة شهرياً مقابل الحد المسموح
- 💵 **متابعة الرصيد الحالي** - عرض رصيد كل محفظة
- 🎨 **تلوين مميز** - كل محفظة لها لون خاص للتمييز السهل
- 📝 **سجل المعاملات** - تسجيل جميع عمليات الإرسال والاستلام
- ⚡ **تحديث فوري** - تحديث الإحصائيات تلقائياً بعد كل معاملة
- 📱 **واجهة عربية** - تصميم كامل بالعربية مع دعم RTL
- 🎯 **مؤشرات بصرية** - ألوان تحذيرية عند اقتراب الحد (أخضر، أصفر، أحمر)

## 🚀 كيفية الاستخدام

### 1. إضافة محافظ جديدة

اضغط على زر **"+ إضافة محفظة جديدة"** في الأعلى وأدخل:
- اسم المحفظة (مثال: محفظة رقم 1)
- رقم الهاتف
- الحد اليومي (بالجنيه المصري)
- الحد الشهري (بالجنيه المصري)
- الرصيد الحالي
- لون مميز للمحفظة

### 2. إضافة معاملات

اضغط على **"+ إضافة معاملة"** لتسجيل:
- **إرسال (خصم)**: عند إرسال أموال من المحفظة
- **استلام (إضافة)**: عند استلام أموال في المحفظة

كل معاملة تؤثر على:
- الرصيد الحالي
- الحد اليومي المستخدم
- الحد الشهري المستخدم

### 3. عرض تفاصيل المحفظة

اضغط على أي محفظة لعرض:
- جميع معاملات المحفظة
- تفاصيل كل عملية إرسال واستلام
- التاريخ والوقت لكل معاملة

### 4. البيانات التجريبية

عند فتح التطبيق لأول مرة، يمكنك الضغط على زر **"🎲 بيانات تجريبية"** لإضافة 6 محافظ تجريبية تلقائياً.

## 📊 فهم المؤشرات

### شريط التقدم - الألوان:
- 🟢 **أخضر** (0-70%): استخدام طبيعي
- 🟡 **أصفر** (70-90%): تحذير - اقتراب من الحد
- 🔴 **أحمر** (90-100%): خطر - وصول أو تجاوز الحد

### أنواع المعاملات:
- 📤 **إرسال**: خصم من الرصيد ويحسب ضمن الحد اليومي/الشهري
- 📥 **استلام**: إضافة للرصيد ولا يحسب ضمن الحدود

## 🛠️ التقنيات المستخدمة

- **Next.js 16** - إطار عمل React
- **PostgreSQL** - قاعدة البيانات
- **Drizzle ORM** - للتعامل مع قاعدة البيانات
- **Tailwind CSS** - التصميم والتنسيق
- **TypeScript** - لضمان جودة الكود

## 💡 ملاحظات مهمة

1. **الحسابات اليومية**: تُعاد الحسابات تلقائياً كل يوم (من الساعة 00:00)
2. **الحسابات الشهرية**: تُعاد الحسابات تلقائياً في أول كل شهر
3. **الحدود**: يتم حساب فقط المعاملات من نوع "إرسال" في الحدود اليومية والشهرية
4. **الرصيد**: يتأثر بكل من عمليات الإرسال والاستلام

## 🎯 أمثلة على الاستخدام

### مثال 1: تسجيل عملية تحويل
1. اختر المحفظة
2. اختر "إرسال (خصم)"
3. أدخل المبلغ: 500 ج.م
4. أدخل الوصف: "تحويل للعميل أحمد"
5. اضغط "إضافة"

النتيجة:
- الرصيد ينخفض 500 ج.م
- الحد اليومي المستخدم يزيد 500 ج.م
- الحد الشهري المستخدم يزيد 500 ج.م

### مثال 2: تسجيل استلام أموال
1. اختر المحفظة
2. اختر "استلام (إضافة)"
3. أدخل المبلغ: 1000 ج.م
4. أدخل الوصف: "استلام من العميل محمد"
5. اضغط "إضافة"

النتيجة:
- الرصيد يزيد 1000 ج.م
- الحدود اليومية والشهرية لا تتأثر
