# 🚀 البداية السريعة - 5 دقائق

## خطوة واحدة للبدء!

### افتح التطبيق واضغط "🎲 بيانات تجريبية"

سيضيف لك **6 محافظ جاهزة** فوراً! 🎉

---

## أول 5 دقائق

### 1️⃣ افتح التطبيق (دقيقة واحدة)
```
✓ افتح الرابط في المتصفح
✓ ستجد شاشة ترحيب بسيطة
```

### 2️⃣ أضف بيانات تجريبية (10 ثواني)
```
✓ اضغط زر "🎲 بيانات تجريبية"
✓ تم! الآن لديك 6 محافظ جاهزة للتجربة
```

### 3️⃣ استكشف المحافظ (دقيقة واحدة)
```
✓ ستجد 6 محافظ ملونة
✓ كل محفظة تعرض:
  - الرصيد الحالي
  - الحد اليومي مع شريط تقدم
  - الحد الشهري مع شريط تقدم
```

### 4️⃣ جرّب إضافة معاملة (دقيقتان)
```
✓ اضغط "+ إضافة معاملة"
✓ اختر أي محفظة
✓ اختر "إرسال (خصم)"
✓ أدخل مبلغ: 500
✓ اضغط "إضافة"
✓ شاهد التحديث الفوري! ⚡
```

### 5️⃣ اضغط على محفظة (دقيقة واحدة)
```
✓ اضغط على أي بطاقة محفظة
✓ ستظهر معاملات هذه المحفظة فقط
✓ اضغط "عرض الكل" للعودة
```

---

## ✅ الآن أنت جاهز!

في 5 دقائق فقط تعلمت:
- ✓ كيف تضيف محافظ
- ✓ كيف تسجل معاملات
- ✓ كيف تتابع الحدود
- ✓ كيف تراجع السجلات

---

## 🎯 الخطوة التالية؟

### أضف محافظك الحقيقية:

1. **اضغط "+ إضافة محفظة جديدة"**

2. **أدخل البيانات:**
   ```
   اسم المحفظة: محفظة العمل
   رقم الهاتف: 01012345678
   الحد اليومي: 5000
   الحد الشهري: 50000
   الرصيد الحالي: 2500
   اللون: (اختر لون مفضل)
   ```

3. **ابدأ التسجيل:**
   - سجل كل عملية إرسال
   - سجل كل عملية استلام
   - راقب الحدود باستمرار
